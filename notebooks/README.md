# Notebooks

This folder contains Jupyter/Colab notebooks used during the project.

## Main Notebook
### preprocessing.ipynb
This notebook includes:
1. Loading the dataset.
2. Inspecting missing values.
3. Dropping columns with high missing percentages.
4. Imputing missing values.
5. Selecting numeric columns.
6. Calculating Z-scores and removing outliers.
7. Saving cleaned datasets to `data/processed`.

## Notes
- The notebook is organized step-by-step following the assignment tasks.
- Outputs (tables, shapes, summaries) appear under their respective sections.
